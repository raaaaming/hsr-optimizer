# HSR Optimizer

Honkai: Star Rail Build Optimizer & Battle Simulator

## Features

- Character Build Manager
- Party Manager
- Cycle Editor
- Battle Simulator
- Damage Engine
- Relic Optimizer
- Mobile UI
- Cloudflare Workers

## Development

Install

```bash
npm install
```

Run locally

```bash
npm run dev
```

Deploy

```bash
npm run deploy
```

## Stack

- Cloudflare Workers
- Vanilla JavaScript (ES Modules)
- HTML
- CSS
- No Framework
